# Documentación de Cal.com

Bienvenido a la documentación de Cal, donde encontrarás consejos y explicaciones para todas las funciones y servicios de nuestro producto. También encontraras tips y trucos para sacarle el mejor provecho a tu suscripción.

Aquí también vas a encontrar documentación acerca de nuestro sistema de diseño, como asi también documentación de soporte para el desarrollador.

Querés colaborar en mejorar esta documentación? Cal es un software de código abierto y esta disponible en [GitHub](https://github.com/calendso/docs). Al final de cada página verás un botón para editar el código fuente y enviar una PR (pull request).
